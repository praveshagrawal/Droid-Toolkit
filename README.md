 - Root or Unroot
 - Change Device ID
 - Unlock Bootloader
 - Install Custom Recovery
 - Remove Pattern Lock
 - Hard Reset
 - Custom BootAnimation
 - Install or Uninstall Apps
 - OTA Update
 - ADB Shell
 - Data Backup
